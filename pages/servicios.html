Sala de extraccion

El alquiler de una sala de extraccion de miel es un servicio que ofrecemos para que las personas puedan nuestros extractores de miel. La sala de extraccion de miel está equipada con todo lo necesario para la extracción de miel, y el personal de la granja estará disponible para ayudar a los clientes durante el proceso.

El equipo para extraer la miel suele constar de un cajón de entrada, una cuba de extracción, una caldera, un espinazo y una espumadera. La cuba de extracción es una gran tina metálica que se llena con las colmenas. El cajón de entrada se usa para depositar las colmenas una vez que se han vaciado. La caldera se usa para hervir la miel y eliminar impurezas. El espinazo se usa para filtrar la miel y la espumadera se usa para quitar la espuma de la miel.

Retiro de enjambres de abejas

El servicio de retiro de enjambres de abejas consiste en el envío de un apicultor a tu propiedad para retirar un enjambre de abejas. El apicultor suele utilizar una caja de enjambre para atrapar las abejas y llevarlas a su granja, donde se les permite vivir y trabajar.
